# Mapeamento Corporal

Scripts de rastreamento e interação com o corpo usando MediaPipe.

## Estrutura
- `face/`: Detecção e análise facial.
- `body/`: Mapeamento de corpo, braços e mãos.
- `interactive/`: Controle de objetos e efeitos interativos.

## Como usar
Execute cada script com `python <arquivo>.py` (necessita webcam).
